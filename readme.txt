== Frequently Asked Questions ==

= Is the plugin free to use? =
Yes. SmartWrite AI is free and open source. You just need your own OpenAI API key.

= How do I get an OpenAI API key? =
1. Visit https://platform.openai.com/signup  
2. Go to https://platform.openai.com/api-keys  
3. Click "Create new secret key"  
4. Paste it into SmartWrite AI settings

= How much does it cost to use? =
Pricing depends on usage. SmartWrite uses GPT-3.5 Turbo by default:

- Intro paragraph: ~$0.01  
- Summary: ~$0.02  
- Meta description: <$0.01  

OpenAI charges per use. Monitor usage at https://platform.openai.com/account/usage

= Does this plugin store or share my data? =
No. The plugin sends prompts and options directly to OpenAI. No data is stored or logged by SmartWrite AI.
