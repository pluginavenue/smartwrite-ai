== Frequently Asked Questions ==

= Is the plugin free to use? =
Yes. SmartWrite AI is free and open source. You just need your own OpenAI API key.

= How do I get an OpenAI API key? =
1. Visit https://platform.openai.com/signup and create an account  
2. Go to https://platform.openai.com/api-keys  
3. Click "Create new secret key" and copy it  
4. Paste the key into SmartWrite AI settings

= How much does it cost to use? =
That depends on how much content you generate. GPT-3.5 Turbo is the default model and is very affordable:

- Intro paragraph: ~$0.01  
- Summary: ~$0.02  
- Meta description: <$0.01

OpenAI only charges per usage. You can track your usage here: https://platform.openai.com/account/usage

= Does this plugin send data to third-party servers? =
Only your prompt and settings are sent to OpenAI via their API. No data is stored or shared by the plugin.
