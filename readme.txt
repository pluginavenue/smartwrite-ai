== Frequently Asked Questions ==

= Is the plugin free to use? =
Yes, SmartWrite AI is free. However, you must provide your own OpenAI API key. OpenAI charges per use, with no fixed monthly fee unless you opt into one.

= How do I get an API key? =
1. Go to https://platform.openai.com/signup and create an account
2. Once logged in, go to https://platform.openai.com/api-keys
3. Click "Create new secret key" and copy the key
4. Paste your key into the SmartWrite AI settings page

= How much does it cost to use? =
Pricing depends on how much content you generate. The plugin uses the `gpt-3.5-turbo` model by default for its balance of speed and low cost. Example costs:

- Intro paragraph: ~$0.01  
- Post summary: ~$0.02  
- Meta description: <$0.01

You can monitor and set usage caps on your OpenAI billing dashboard: https://platform.openai.com/account/usage
